# CleanTimer

[CleanTimer.io](https://cleantimer.io) - A Clean and Simple Timer App

![Screenshot](./screenshot.png)

## Requirements
- *nix Operating System
- NodeJS

## Building
```bash
> npm i
> ./build.sh
```

## Development
```bash
> brew install fswatch && npm i -g http-server typescript
> ./watch.sh & http-server
```
